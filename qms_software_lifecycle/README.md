# QMS for Software Lifecycle

This Odoo 19 Community Edition addon provides a structured, auditable Quality Management System (QMS) tailored for software companies.

## Features

- **Requirements Management**: Track business, functional, non-functional, and compliance requirements.
- **Design & Specification**: Manage high-level and detailed designs linked to requirements.
- **Test Management**: Define test plans and test cases, execute test runs, and track results.
- **Defect Tracking**: Full lifecycle for software defects with root cause analysis and CAPA.
- **Release Management**: Plan and track software releases with included features and resolved issues.
- **Change Management**: Formal workflow for change requests with impact analysis.
- **Risk Management**: Identify and mitigate technical, organizational, and security risks.
- **Document Management**: Versioned QMS documents, including User Manuals.
- **ISO 9001 Alignment**: Designed to provide the documented evidence required for ISO 9001 certification.

## Installation

1. Copy the `qms_software_lifecycle` folder to your Odoo addons directory.
2. Update the app list in Odoo.
3. Install the "QMS for Software Lifecycle" module.

## Usage

Access the QMS features through the "Quality" top-level menu.

### Roles

- **QMS Manager**: Full access to all QMS features and configuration.
- **QMS User**: Create and manage QMS artifacts (requirements, tests, etc.).
- **QMS Read-Only (Auditor)**: Read-only access for auditing and compliance checks.
